6. Wireshark

By Akshay Chavan
Posted 2023-11-30 1 min read

https://www.lifewire.com/wireshark-tutorial-4143298

Introduction

Originally know as Ethereal , Wireshark display data from hundreds of different protocols on all major network types.

Capture data packets

  1. Select one or more network interface, go to menu bar then select Capture.

[!tip] To select mutiple network interface , hold the Shift key as you make your selection

  1. In the Wireshark Capture Interface window , select Start .

  2. Select File > Save As or choose an Export option to record the capture

  3. To stop capturing, press Ctrl+E. Or, go to the Wireshark toolbar and select the red Stop button that’s located next to the shark fin.

Analyze Packet Contents

We focus on three main Sections :

  1. Packet list pane (top section)
  2. Packet details pane (middle section)
  3. packet bytes pane (bottom section)

Packet List

  • located at top windows, show all packet found in the active capture file.

| columns | Description | | ———– | ———————————————————————————————————————————- | | No | This field indicates which packets are part of the same conversation. It remains blank until you select a packet. | | Time | The timestamp of when the packet was captured is displayed in this column | | Source | the address (IP or other) where the packet originated. | | Destination | the address that the packet is being sent to. | | Protocol | The packet’s protocol name, such as TCP, can be found in this column. | | Length | The packet length, in bytes | | Info | Additional details about the packet are presented here. The contents of this column can vary greatly depending on packet contents. |

Packet Details

  • Presents the protocols and protocol fields of the selected packet.

Packet Bytes

  • display the raw data of selected packet in a hexadecimal view.
